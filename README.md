# Text Messenger Server

Простой и безопасный сервер для мессенджера.

## Как запустить

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите: `python app.py`

## API

- `POST /register` - регистрация
- `POST /login` - вход
- WebSocket - онлайн сообщения
